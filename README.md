# Smart City Weather Dashboard

A modern weather dashboard built with HTML, CSS, and JavaScript. It fetches live weather data and a 5-day forecast for any city using a public API.

## Features
- City search
- Current weather display
- 5-day forecast
- Forecast sorting by temperature
- Dark/light theme toggle
- Responsive premium UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
- Open-Meteo API

## JavaScript Concepts Used
- Async/Await
- DOM Manipulation
- Array map()
- Array filter()
- Array sort()
